## Publishing checklist (for each new post)

1. [ ] Write to `src/content/journal/_drafts/my-slug.md` — Carlos reviews
2. [ ] Add all 8 frontmatter fields including `created_by`, `market`, `funnel_stage`, `content_goal`
3. [ ] Move to `src/content/journal/my-slug.md` when approved
4. [ ] Run `node scripts/render-journal-og.mjs` → set `image:` in frontmatter
5. [ ] Add `/journal/my-slug` to `reactSnap.include` in `package.json`
6. [ ] Add `<url>` block to `public/sitemap.xml`
7. [ ] Compliance check (see `_drafts/_WORKFLOW.md`)
8. [ ] Commit and push → Netlify deploys in ~60 seconds
